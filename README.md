# Machine Learning - Téo Me Why (2025)

Bem-vindo(a) ao repositório de atividades práticas do curso de **Machine Learning** do canal **Téo Me Why**. Aqui você encontrará os códigos, scripts e experimentos realizados ao longo do curso.

## 🗂️ Estrutura do Repositório

O repositório está organizado da seguinte forma:

- **`semana_01/`**: Fundamentos de Machine Learning.
  - Scripts básicos abordando conceitos de `classificacao.py`, `regressao.py` e avaliação de modelos (`metricas.py`).
- **`churn/`**: Projeto prático de predição de Churn.
  - Contém o script de treinamento de modelo (`train.py`) e o script de inferência/predição (`predict.py`).
- **`data/`**: Diretório reservado para os conjuntos de dados (datasets) utilizados durante o curso.

## 🛠️ Tecnologias Utilizadas

Algumas das principais tecnologias e bibliotecas trabalhadas neste repositório:
- Python
- Scikit-Learn
- Pandas
- MLflow

## 📚 Sobre o Curso

Este material faz parte das aulas práticas da comunidade e canal [Téo Me Why](https://www.youtube.com/@teomewhy) focadas em democratizar o acesso à área de dados e Machine Learning no Brasil.
